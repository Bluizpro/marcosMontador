## Dicas para Usar os Depoimentos

### 1. **Adicionar Fotos dos Clientes**
Para aumentar a credibilidade, você pode adicionar fotos dos clientes ao lado de cada depoimento. Adicione uma tag `<img>` antes ou depois do texto do depoimento.

### 2. **Adicionar Datas**
Considere adicionar datas para mostrar que os depoimentos são recentes. Exemplo: "Há 2 meses" ou "Janeiro de 2026".

### 3. **Diversificar os Depoimentos**
Os depoimentos atuais cobrem:
- ✅ Montagem residencial (guarda-roupa, cozinha)
- ✅ Montagem corporativa (escritório)
- ✅ Desmontagem e mudança
- ✅ Manutenção e reparo

### 4. **Adicionar Mais Depoimentos**
Conforme você receber mais avaliações de clientes, adicione-as à página. Mantenha sempre os 5-6 melhores depoimentos visíveis.

### 5. **Usar Redes Sociais**
Incentive seus clientes a deixarem avaliações no Google, Facebook e Instagram para aumentar a confiança.

## Elementos-Chave dos Depoimentos

Cada depoimento foi construído com estes elementos persuasivos:

| Elemento | Descrição | Exemplo |
|----------|-----------|---------|
| **Especificidade** | Menciona o tipo de serviço realizado | "guarda-roupa planejado", "cozinha planejada" |
| **Detalhe Pessoal** | Inclui detalhes que mostram atenção | "deixou tudo impecável", "deixou o local limpo" |
| **Repetição** | Mostra que o cliente contratou mais de uma vez | "Já chamei ele mais 2 vezes" |
| **Recomendação** | Indica que o cliente recomenda para outros | "Recomendo para todos", "Já recomendei para vários amigos" |
| **Qualidade + Preço** | Equilibra qualidade com valor | "preço justo", "preço bem acessível" |
| **Profissionalismo** | Destaca a conduta profissional | "muito profissional", "discreto" |

## Como Personalizar para Seu Negócio

Se você quiser adicionar depoimentos reais de seus clientes, siga este template:

```
**Classificação:** ⭐⭐⭐⭐⭐ (5 estrelas)

**Texto:**
"[Depoimento do cliente com detalhes específicos do serviço]"

**Cliente:** [Nome do Cliente]
**Localização:** [Cidade, Estado]
**Tipo de Serviço:** [Qual serviço foi realizado]
```

Mantenha os depoimentos com 2-3 linhas para melhor visualização na página.
